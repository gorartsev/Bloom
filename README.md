# BLOOM — Дневник тренировок 🌸

Милое приложение-календарь для записи тренировок. Розовая тема, минималистичный дизайн.

## Что умеет
- Ввод имени при первом запуске (персонализированное приветствие)
- Месячный календарь с отметками тренировок
- Добавление тренировки на любую прошедшую дату или сегодня
- 10 типов тренировок (кардио, силовая, йога, пилатес, танцы, растяжка и т.д.)
- Длительность, время, настроение, заметки
- Счётчик дней подряд 🔥
- Статистика: график за 7 дней, любимые типы тренировок
- Всё хранится локально на устройстве

## Как разместить на GitHub Pages

### Шаг 1: Создать репозиторий
1. Зайди на github.com → "+" → "New repository"
2. Название: `bloom` (или любое)
3. Public → Create

### Шаг 2: Загрузить файлы
1. На странице репо → "uploading an existing file"
2. Загрузи все 5 файлов:
   - `index.html`
   - `manifest.json`
   - `icon-180.png`
   - `icon-192.png`
   - `icon-512.png`
3. Commit changes

### Шаг 3: Включить Pages
1. Settings → Pages
2. Source: Deploy from a branch
3. Branch: main, папка /(root)
4. Save
5. Через 1-2 минуты будет доступно на `https://ТВОЙ_ЛОГИН.github.io/bloom/`

### Шаг 4: Добавить на iPhone
1. Открой ссылку в **Safari**
2. Поделиться → "На экран Домой"
3. Назови BLOOM → Добавить

Готово! Приложение работает офлайн, данные сохраняются локально.
